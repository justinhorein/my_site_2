Django Movie Manager
   A web app I built as part of an internship for Prosper I.T. Consulting. The application queries a Movie Database API (IMDB Alternative) and displays results accordingly. Along with the ability to search for movies, you can store titles you have already seen or own into a database (denoting details like Year Released and The Platform you have access to it on). It's here to help organize your Film Watching experience.
